# MCP Git Commit Tools

Small MCP server exposing git-related tools for Warp (or any MCP-enabled LLM client)
to read the staged diff and changed files so the LLM can generate commit messages. mcp-warp-github-commit


# Install MCP Server locally
```bash
git clone https://github.com/raiashpanda007/mcp-warp-github-commit/
cd mcp-warp-github-commit
npm i
tsc

```

## Open Your warp terminal > Search MCP Command :: node and args as dist/index.js
